# Portfolio
A modern, fully responsive personal portfolio website showcasing professional experience, skills, projects, and research publications. Built with HTML5, CSS3, and JavaScript with dark mode support and smooth animations.

---

## 🎯 Features

### ✨ Core Features
- **Responsive Design**: Fully mobile-optimized with breakpoints at 768px and 480px
- **Dark Mode Toggle**: Seamless theme switching between light and dark modes
- **Smooth Animations**: AOS (Animate On Scroll) library for engaging transitions
- **Mobile Navigation**: Hamburger menu with full-screen overlay for mobile devices
- **Sticky Navigation**: Fixed top navigation bar for easy access to all sections
- **Keyboard Navigation**: Accessible scrolling with smooth behavior
